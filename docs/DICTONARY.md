# Dicionário:

> #### ℹ️ _Base_:
> `Base` são valores padrão, mas podem receber `modificadores`.

> #### ℹ️ _Modificadores_:
> Valores que transformam os atributos.

> #### ℹ️ _Intervalos_:
> `Intervalos` serão usados para gerar as recompensas do jogador quando o inimigo for morto.

> #### ℹ️ _Estado_:
> Itens Equipáveis (espadas, escudos, armaduras, etc.) possuem `Estado` para indicar se estão sendo usados pelo Jogador;
>
> Um equipamento em uso não pode ser equipado sobre si.
